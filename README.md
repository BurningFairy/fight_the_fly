"# SYP---Pygame---FIGHT-THE-FLY" 

You play as Berni the Apple.
Your goal is to survive and defeat flying enemies (flies) using different weapons.

The game has:

A start menu
A settings menu
A gameplay screen
Enemies that follow the player
Melee and ranged weapons with auto-fire

How to play:
Move the player with the keyboard
The character attacks automatically
Enemies follow the player
When all enemies are defeated, new enemies spawn.

Controls:

W / A / S / D or Arrow Keys → Move player

Weapons fire automatically in the last movement direction
Mouse Click → Navigate menus
Close Button → Return to menu

Requirements:

Python 3.9 (Download from https://www.python.org)
Pygame (pip install pygame)
OS: Windows 11
(If you use Linux as your OS, you need Wine.)

Instructions:

To play the game open the "Dis" folder and double-click the main.exe file
